## 📌 Project Overview  

This project focuses on analyzing **bank loan data** to identify customers who are likely to **default on their loans**.  

📊 The goal is to help financial institutions make **data-driven decisions** and reduce risk.  

---

## 🎯 Objectives  

✔️ Identify high-risk customers  
✔️ Analyze factors affecting loan default  
✔️ Perform data cleaning and preprocessing  
✔️ Extract actionable insights  
✔️ Visualize patterns and trends  

---

## 🧠 Problem Statement  

Banks face significant losses due to loan defaults.  

👉 The challenge is to:  
- Detect risky applicants  
- Understand key influencing factors  
- Improve loan approval decisions
